#### 4.13
++i because,
```
#include <stdio.h>

int main(void) {
	int i, j, k;
	
	i = 5; j = 5; k = 5;
	printf("4.13: %d %d %d \n", ++i, j++, k+=1 );
	
	return 0;
}
```
outputs:
```
4.13: 6 5 6
```
